CODLE RULES

  • There is a set list of 5-letter terms in relation the JavaScript, HTML, and CSS
  • The game will select a term at random and you have to guess the term!
  • Click the letters on the on-screen keyboard to spell out the term.
  • Press enter to submit the guess and press delete if you clicked the wrong letter.
  • If you got the letter correct, but in the wrong place, the tile will turn Pink.
  • If you got the letter correct and in the correct place, the tile will turn Blue!
